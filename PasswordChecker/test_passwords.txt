Password Strength Test Cases
============================

WEAK PASSWORDS:
- password
- 123456
- admin
- qwerty
- abc123
- password123
- 12345678
- welcome
- login
- guest

MEDIUM PASSWORDS:
- Password1
- MyPass123
- Welcome2024
- Admin@123
- Qwerty456
- Hello123!
- Computer1
- Security2024
- Test@Pass
- User123456

STRONG PASSWORDS:
- MyD0g&Cat!Love2Play
- Tr@v3l!ng*Ar0und&W0rld
- C0ff33&C00k!es#2024
- B00ks&M0v!es#MyL!fe
- Sp0rts&Gam3s*Fun!2024
- Cyb3r$3cur!ty&Educat!0n
- Pr0gr@mm!ng*C0d3&Fun
- N3tw0rk$3cur!ty&Analys!s
- D@t@$c!3nc3&M@ch!n3L3@rn!ng
- Qu@ntum*C0mput!ng&Futur3

EDGE CASES:
- a (too short)
- ALLUPPERCASE (no variety)
- alllowercase (no variety)
- 1234567890 (only numbers)
- !@#$%^&*() (only symbols)
- aaaaaaaaaa (repeated characters)
- abcdefghij (sequential pattern)
- qwertyuiop (keyboard pattern)

COMMON PATTERNS TO AVOID:
- password + numbers (password123, password2024)
- keyboard patterns (qwerty, asdf, 123456)
- dictionary words (computer, internet, security)
- personal info patterns (name + birthyear)
- simple substitutions (@ for a, 3 for e)

INTERNATIONAL CHARACTERS:
- Pássw0rd!@#
- Contraseña123!
- Mötlöy&Crü3!
- 密码Strong123!
- Пароль2024!
